# Heart Disease Prediction 🫀

## 📌 Project Overview
This project predicts whether a person has cardiovascular disease using Machine Learning models.

## 🔧 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib & Seaborn

## 📊 Models Used
- Logistic Regression
- KNN
- Decision Tree
- Random Forest
- SVM

## 📈 Results
Best Accuracy: ~73%

## 🚀 How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn matplotlib seaborn

2. Run the notebook:
   jupyter notebook

## 📁 Dataset
Cardiovascular Disease Dataset
